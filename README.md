# Excel practice files
Practice datasets for the [Dataroos YouTube channel](https://youtube.com/@dataroos) Excel tutorials (starting from EP012).

One folder per video — download the file, open the matching video, follow along.

### ⚠️ Please note:
- All data is fictional and generated for teaching. Names, companies, and figures are invented; any resemblance to real people or businesses is coincidental.
- The files contain deliberate errors — that's the point of the tutorials.
### How to download a practice file?
In the description section of a YouTube video, click the **Practice File** link and you will be directed to my GitHub repository. 
For example, in video EP012 [XLOOKUP Has 5 Features Nobody Shows You](youtube/@dataroos), after you click the [**Practice File** link](https://github.com/dataroos/excel-practice-files/blob/main/EP012/XLOOKUP.xlsx), you can click the download button as indicated by the $\color{rgb(255,0,0)}{\textsf{RED}}$ arrow in the screenshot below.\
