# excel-practice-files
Practice datasets for the [Dataroos](https://youtube.com/@dataroos) Excel tutorials.

One folder per video — download the file, open the matching video, follow along.


⚠️ All data is fictional and generated for teaching. Names, companies, and figures are invented; any resemblance to real people or businesses is coincidental.
  The files contain deliberate errors — that's the point of the tutorials.
